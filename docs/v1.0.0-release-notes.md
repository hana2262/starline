# StarLine v1.0.0 Release Notes

## Summary

This release marks the first MVP release candidate for StarLine.

It includes:
- Windows local-first desktop runtime
- project and asset management
- MiniMax and Stable Diffusion connector support
- retrieval-backed Agent workspace
- local analytics dashboard
- desktop display language selection

## v1.0.0 Patch Additions

This patch adds:
- display language selector in the desktop header
- support for:
  - Simplified Chinese
  - English
- localized desktop navigation
- localized project, asset, connector, agent, and analytics pages
- persisted language preference through local storage
