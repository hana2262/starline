# StarLine

Windows local-first AI creator workspace built with Tauri, React, Fastify, and SQLite.

Current MVP capabilities:
- project CRUD
- local asset import and retrieval
- MiniMax and Stable Diffusion connector management
- async generation jobs with retry, recovery, cancel, and metrics
- retrieval-backed agent sessions
- local analytics events, APIs, and desktop dashboard

## Workspace

- desktop UI: `apps/desktop-ui`
- local API: `apps/local-api`
- shared packages: `packages/*`
